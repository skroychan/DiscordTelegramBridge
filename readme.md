a simple two-way Telegram-Discord bridge

made with [pyTelegramBotAPI](https://github.com/eternnoir/pyTelegramBotAPI) and [discord.py](https://core.telegram.org/bots/api)
- - -
supported features:
- [x] replies
- [x] media
- [x] media embeds (to some extent)
- [x] stickers (except vector)
- [x] custom discord emojis
- [ ] system messages
- [ ] forwarded messages
- [ ] user aliases
- [ ] multiple bridges
- [ ] custom telegram emoji
- [ ] markdown
